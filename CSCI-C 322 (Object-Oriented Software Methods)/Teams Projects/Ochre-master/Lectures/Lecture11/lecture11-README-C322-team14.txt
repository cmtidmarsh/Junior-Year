Singleton Pattern:
	Advantages:
		- Easy to implement
		- Easy to understand
	Disadvantages:
		- May not be the proper use case for all cases

Global Variable:
        Advantages:
		- Easy to implement
	Disadvantages:
		- Modified and read by all

Command Pattern:
	Advantages:
		- Breaks problem down
		- More robust
	Disadvantages:
		- Possibly harder to implement
		- More moving parts

Lambda Expression:
        Advantages:
		- General usage
	Disadvantages:
		- Can be confusing for people new to programming
